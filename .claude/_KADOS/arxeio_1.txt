Τρία βασικά αίτια, με συγκεκριμένες διορθώσεις:

Δεν εφαρμόζεται το CSS που διορθώνει το “κενό”
– Το index.css μηδενίζει html, body, #root { margin:0 } και ορίζει full-screen, αλλά αν δεν έχει εισαχθεί πρώτο στο entry (π.χ. main.tsx), ο browser κρατά το default margin και βλέπεις τον χάρτη να «μπαίνει» πάνω από την επικεφαλίδα. Επιπλέον περιέχει ακραίες ρυθμίσεις position: fixed για html, body, #root που μπλοκάρουν το position: sticky του header. 

index


Διόρθωση:

Βάλε το import './index.css' πρώτο στο entry.

Αφαίρεσε το position: fixed από html, body, #root και κράτα μόνο:

html, body, #root { margin:0; padding:0; height:100%; overflow:hidden; }


Το sticky θα δουλέψει κανονικά και το κενό θα εξαφανιστεί.

Header “τεράστιος”, user-icon όχι δεξιά, «Ακίνητα/Εργασία» δεν μπαίνουν στη σειρά
– Το component αποδίδει utility κλάσεις τύπου layera-flex, layera-flex--space-between, layera-flex--gap-*, αλλά αυτές οι κλάσεις δεν υπάρχουν μέσα στο index.css. Άρα το layout πέφτει στο default block flow: μεγάλα default margins σε h1/p, στοιχεία στοιβάζονται κάθετα, και τα actions δεν πάνε δεξιά. 

LayeraHeader


Διόρθωση: πρόσθεσε τις βασικές utility κλάσεις ή φόρτωσε το σωστό stylesheet του layout πριν από το app. Ελάχιστο set:

.layera-flex{ display:flex }
.layera-full-width{ width:100% }
.layera-flex--align-center{ align-items:center }
.layera-flex--space-between{ justify-content:space-between }
.layera-flex--gap-sm{ gap: .5rem } /* ή var(--la-space-sm) αν υπάρχει */
h1, p{ margin:0 } /* κόψε τα default margins που φουσκώνουν το header */


Τα navigation buttons έχουν δικό τους selector .layera-nav-button, αλλά πρέπει να εφαρμόζονται πάνω στα στοιχεία. Αν αποδίδεις σκέτα <button>, δώσε τους αυτή την κλάση. 

index

Μη αντιστοίχιση “fixed” vs “sticky”
– Στο CSS υπάρχει .layera-header--fixed { position: fixed; top:0; z-index:1000; }, όμως το LayeraHeader δεν βάζει αυτή την κλάση. Έχει inline position:'sticky' με zIndex:10. Αν ο χάρτης ή wrapper έχει z-index:1 και ο header είναι sticky αλλά το page container είναι fixed, προκύπτει οπτική ασυνέπεια.
Διόρθωση: αποφάσισε ένα μοντέλο. Προτείνω fixed:

Στο LayeraHeader βγάλε το inline sticky και πέρασε className="layera-header--fixed ...".

Πρόσθεσε padding-top στο κύριο wrapper ίσο με το ύψος header:

.layera-map-app{ padding-top: var(--header-height, 64px); }


ή αν κρατήσεις sticky, τότε αφαίρεσε το position: fixed από τα html/body/#root όπως στο (1).

CSS variables χωρίς tokens
– Το index.css χρησιμοποιεί μεταβλητές var(--la-space-*). Αν το tokens.css δεν φορτωθεί πριν, οι μεταβλητές λείπουν και κανόνες απορρίπτονται.
Διόρθωση: φόρτωσε πρώτο το tokens.css (ή @layera/tokens/css) και μετά το index.css.
