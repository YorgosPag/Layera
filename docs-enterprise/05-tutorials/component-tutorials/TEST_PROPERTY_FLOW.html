📋 ΟΔΗΓΙΕΣ ΔΟΚΙΜΗΣ

  1. Κλικ στο πράσινο FAB → Τι εμφανίζεται;
  2. Επιλογή "Ακίνητα" → Ποιο είναι το επόμενο βήμα;
  3. Κάθε κλικ → Καταγραφή του αποτελέσματος
  4. Συνέχεια μέχρι το τέλος → Όλη η ροή ακινήτων

🌐 ΕΦΑΡΜΟΓΗ

📝 ΚΑΤΑΓΡΑΦΗ ΒΗΜΑΤΩΝ

ΠΑΡΑΤΗΡΗΣΗ:
  • Χάρτης φορτώνει
  • Πράσινο FAB button διαθέσιμο

🎯 ΒΗΜΑ 2: ΚΛΙΚ ΣΤΟ FAB

ΕΝΕΡΓΕΙΑ: Κλικ στο πράσινο FAB
ΑΠΟΤΕΛΕΣΜΑ:

🎯 ΒΗΜΑ 3: ΕΠΙΛΟΓΗ ΑΚΙΝΗΤΩΝ

ΕΝΕΡΓΕΙΑ: Κλικ στο "Ακίνητα"
ΑΠΟΤΕΛΕΣΜΑ:

🎯 ΒΗΜΑ 4+: ΣΥΝΕΧΕΙΑ ΡΟΗΣ

ΠΑΡΑΤΗΡΗΣΕΙΣ:

🌳 ΑΠΟΤΕΛΕΣΜΑ - ΔΕΝΔΡΟΕΙΔΗΣ ΔΟΜΗ
