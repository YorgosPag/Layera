# Layera

Μια σύγχρονη εφαρμογή web με Firebase Authentication.

## Δομή Project

```
layera/
  apps/
    layera-id/      # Κύρια εφαρμογή React
  docs/             # Τεκμηρίωση
```

## Εγκατάσταση

```bash
cd apps/layera-id
npm install
```

## Εκκίνηση Development Server

```bash
cd apps/layera-id
npm run dev
```

## Τεχνολογίες

- React 18
- Vite
- Firebase Authentication
- React Router v6

## Firebase Configuration

Το project χρησιμοποιεί Firebase για authentication με το project ID: `layera-dev`
